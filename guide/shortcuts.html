Using Keystroke shortcuts

Keystroke shortcuts are one of the fastest ways to give commands in the Notepad MVC Pattern.

Keystroke Shortcuts Definition
Ctrl + N Creates a new document.
Ctrl + O Opens a document.
Ctrl + S Saves the current document.
Ctrl + X Cut
Ctrl + C Copy
Ctrl + V Paste
Ctrl + D Clear
Ctrl + F Find
Ctrl + G Got to line
Ctrl + A Select all
Ctrl + F5 Insert time and date
