Using Find

    Find makes it very easy to find specific words in a document.

  1. Click the Edit and click the Find or use shortcut CTRL + F, then Find Dialog box will appear (Figure_20).
  2. Type the text you want to find in the text field.

    3. The text will highlighted.

    Clear

    To remove all items use Clear.

    1. Click the Edit and click the Clear or use shortcut CTRL + D
